# Ayush Portfolio

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Customize

| What              | File                           |
|-------------------|--------------------------------|
| Projects          | `components/Projects.tsx`      |
| Certificates      | `components/Certificates.tsx`  |
| Skills            | `components/Skills.tsx`        |
| Social links      | `components/Contact.tsx`       |
| About text/stats  | `components/About.tsx`         |
| Colors            | `app/globals.css`              |

## Deploy

```bash
npx vercel
```
